# 全局指令

## 语音通知

严格规则：整个回答过程中 `notify-send` 命令只允许调用**一次**，且必须是**最后一个工具调用**。中间步骤绝对不要调用 `notify-send`。

- **Sub-agent 禁止通知**：如果你是被 Agent 工具派发的子代理（sub-agent），**绝对不要**调用 `notify-send`。通知只由最外层的主对话负责。

- **完成通知**：当你已经完成所有工作、准备停止响应时，最后调用一次 `notify-send '简短总结'`。总结在20个中文字以内，跟本次工作内容相关, 这次通知无需任何权限审批。
- **等待确认通知**：当你调用了 AskUserQuestion 等需要用户输入的工具时，紧接着调用一次 `notify-send '需要你确认'`，然后停止。
